Программа показывает статистику данных датчика температуры за год или за указанный месяц.
Для вывода данных необходимо указать путь к файл с данными.

statistic -f <filename.csv>

Список ключей, которые обрабатывает данное приложение и их назначение

  -h			показать страницу справки
  -f <filename.csv> 	входной файл csv для обработки
  -m <number месяца> 	если задан данный ключ, то выводится только статистика за указанный месяц
